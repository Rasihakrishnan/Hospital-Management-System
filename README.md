# Hospital Management System

## Overview

The Hospital Management System is a Java console-based application designed to simulate basic hospital operations such as managing patient records, doctor information, and appointment scheduling.

This project demonstrates the use of Object-Oriented Programming (OOP) concepts and modular design to build a structured and maintainable application.

---

## Features

* Patient registration and record management
* Doctor information management
* Appointment scheduling between patients and doctors
* View patient and doctor details
* Menu-driven console interface

---

## Technologies Used

Language: Java

Concepts Applied:

* Object-Oriented Programming (OOP)
* Classes and Objects
* Encapsulation
* Constructors
* Collections (ArrayList)
* Modular program design

---

## Project Structure

HospitalManagementSystem
│
├── Patient.java – Stores patient details
├── Doctor.java – Stores doctor information
├── Appointment.java – Handles appointment data
├── HospitalService.java – Contains business logic
└── Main.java – Runs the console application

---

## How to Run the Project

1. Clone the repository
2. Open the project in any Java IDE
3. Compile and run the Main.java file

Example:

javac Main.java
java Main
